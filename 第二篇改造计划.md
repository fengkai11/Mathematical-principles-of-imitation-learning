# 第二篇改造计划

> 本文档用于指导《模仿学习的数学原理》第二篇：序列决策与轨迹分布基础的成书级改造。
>
> 改造依据：`成书级章节改造方法.md` 与 `公式格式说明.md`。

---

## 一、第二篇范围确认

第二篇为：

```text
第二篇：序列决策与轨迹分布基础
```

包含第 5–9 章：

```text
第5章 MDP：模仿学习虽然不要 reward，但也逃不出序列决策
第6章 从单步损失到轨迹损失：别只盯着眼前那一小步
第7章 确定性策略与概率策略：机器人不要只会一个标准答案
第8章 隐变量策略：给机器人一个动作风格旋钮
第9章 CVAE：训练时偷看答案，推理时假装自己懂了
```

这一篇在全书中的作用是：把第一篇的 BC / 分布偏移 / DAgger，推进到 MDP、轨迹分布、概率策略、隐变量和 CVAE，为后续 IRL、GAIL、ACT、Diffusion Policy、Flow Matching 打基础。

## 二、审阅后的主要问题

### 1. 内容深度基本够，但公式格式明显落后于新版规范

第5章到第9章已经有较多工程例子和公式讲解，尤其第9章 CVAE 内容较厚。但当前大量公式仍使用旧写法：

```markdown
<span class="math">...</span>
<div class="math">\[...\]</div>
```

这与 `公式格式说明.md` 中当前要求不一致。后续改造应统一为：

```markdown
行内公式使用 $...$
独立公式使用 $$...$$
不要让 = 单独占一行
不要使用 \operatorname 或 \operatorname*
范数使用 \lVert ... \rVert
不要把块级公式放进 blockquote 或 Markdown 表格
```

### 2. 公式索引仍然大量使用 Markdown 表格

新版规则要求：公式索引不要把复杂公式塞进 Markdown 表格，而要使用逐条公式索引，并说明含义和掌握程度。

当前第5章和第9章等章节仍使用表格式公式索引，后续需要统一改成：

```markdown
### 公式 (X.Y)：公式名称

$$...$$

- **含义**：这个公式表达什么。
- **需要掌握到什么程度**：读者应该理解到什么层次。
```

### 3. 章节里还残留内部写作说明

第7章、第8章、第9章开头仍有类似：

```text
本章继续遵守 v2.0 总控文档...
```

这类内容属于写作过程说明，不应进入成书正文。后续需要删除，并改写成自然的章节导入。

### 4. 第二篇导读太薄，需要升级

当前第二篇导读能说明范围，但还没有形成“篇级数学主线”。建议升级为：

```text
第一篇解决：专家样本、BC、分布偏移、DAgger
第二篇推进：策略不只是单步函数，而是在 MDP 中诱导轨迹分布
核心数学对象：MDP、p_pi(tau)、J(pi)、pi(a|s)、z、q_phi(z|x,a)、ELBO
后续铺垫：IRL/GAIL 的 occupancy measure，ACT 的 action chunk，Diffusion Policy 的条件动作分布
```

## 三、第二篇改造总目标

第二篇改造的核心不是“写得更多”，而是让读者读完后完成一次认知升级：

```text
从“模仿学习是在专家数据上拟合动作”
升级为
“模仿学习是在序列决策系统中学习能诱导可靠轨迹分布的策略”
```

对应数学主线应收敛为：

```text
MDP
→ 状态、动作、转移
→ 轨迹 tau
→ 策略诱导轨迹分布 p_pi(tau)
→ 单步 loss 与轨迹目标的差异
→ 确定性策略与概率策略
→ 多模态动作分布
→ 隐变量 z
→ CVAE 的 encoder / decoder / ELBO
```

## 四、建议执行顺序

### 第一步：先改第二篇导读

先把导读改厚，不然第5–9章会像独立章节。

导读建议补充：

1. 第二篇为什么出现在第一篇之后；
2. 第二篇新增哪些数学对象；
3. 第5–9章之间的递进关系；
4. 第二篇和第三篇、第四篇的关系；
5. 读完第二篇应该形成哪些工程判断。

### 第二步：逐章改造第5–9章

建议顺序不变：

```text
第5章 MDP
第6章 单步损失到轨迹损失
第7章 确定性策略与概率策略
第8章 隐变量策略
第9章 CVAE
```

每章统一做 6 件事：

1. 删除内部写作说明；
2. 把旧公式格式迁移到 `公式格式说明.md` 要求；
3. 修复公式编号重复、正文公式与索引不一致问题；
4. 将公式索引从表格改为逐条索引；
5. 补齐“读完本章，你应该能判断什么”；
6. 补齐或统一“定义索引、附录阅读、思考题、配图清单、推荐阅读”。

### 第三步：全篇做一致性检查

第5–9章全部改完后，再做一次横向检查：

```text
第5章是否自然引出第6章？
第6章是否自然引出第7章？
第7章是否自然引出第8章？
第8章是否自然引出第9章？
第9章是否自然引出第13章 ACT 和后续生成式策略？
```

## 五、逐章改造重点

| 章节 | 当前定位 | 改造重点 |
|---|---|---|
| 第5章 MDP | 序列决策地基 | 统一 MDP、轨迹分布、return、状态/观测区分；删除旧公式写法；公式索引改逐条 |
| 第6章 单步到轨迹 | 从 step loss 推到 trajectory objective | 强化 $p_\pi(\tau)$ 和 $J(\pi)$；解释 open-loop / closed-loop；修复等号单独占行问题 |
| 第7章 概率策略 | 从确定性动作到条件动作分布 | 强化“条件均值 vs 条件分布”；突出多模态动作和 MSE 平均化问题 |
| 第8章 隐变量策略 | 引入 $z$ 表示隐藏风格/模式 | 明确 $z$ 不是人类标签；强化边缘化、posterior、轨迹级 latent 的区别 |
| 第9章 CVAE | 用 encoder / decoder / ELBO 训练隐变量策略 | 保留深度内容，但重构公式索引；ELBO 推导更适合普通工程师；补齐工程判断 |
