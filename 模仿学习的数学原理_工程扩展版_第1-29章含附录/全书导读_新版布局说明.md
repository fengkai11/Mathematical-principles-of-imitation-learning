# 全书导读：新版布局说明

> 本版将原来的线性章节结构重排为 **八篇、二十九章、九个附录**。调整目标不是推翻原稿，而是把已经扩展出的内容重新组织成一条更清楚的技术演进路线。
>
> 本轮修订进一步明确全书统一主线：**视觉引导机械臂抓取与放置任务**。这条主线会贯穿状态、观测、动作、轨迹、策略、BC、DAgger、ACT、Diffusion Policy、VLA、World Model、Sim-to-Real、OPE 和数据闭环。

## 1. 为什么要重新布局？

随着 Flow Matching、World Models / WAM / JEPA、SSM / Mamba、Preference Alignment / DPO 以及快慢模型被补入，全书已经不再只是一本“行为克隆到 Diffusion Policy”的入门教程，而是扩展为：

```text
模仿学习基础
→ 序列决策与轨迹分布
→ 分布匹配与奖励视角
→ 现代机器人策略模型
→ 长序列与多模态策略架构
→ 世界模型与快慢认知系统
→ 后训练、对齐与实机部署
```

如果继续按追加方式把新内容放在第20章之后，读者会觉得后半部分像“补丁”。新版布局改为分篇结构，目的是让每一组章节都有清晰的问题意识和数学承接。

但仅有分篇还不够。一本数学原理教材还需要一个反复回到的统一例子，否则读者容易觉得自己在读 29 个相关但分散的主题。因此，本版将 **视觉引导机械臂抓取与放置** 设为全书主线：

```text
人类遥操作示范
→ 记录图像、本体状态、夹爪状态和动作
→ 学习观测到动作的策略
→ 闭环执行抓取与放置
→ 从失败、接管、偏好和回放中继续改进
```

这条主线不是为了把本书改成机械臂项目手册，而是为了让每个数学概念都有稳定落点：公式在说什么，符号对应什么工程对象，忽略这个问题会导致哪种机器人失败。

## 3. 本次结构调整做了什么？

本次重排主要做了七件事：

1. 将 **IRL** 前移到 **GAIL** 之前，使第三篇形成“奖励反推 → 分布匹配 → 离线风险”的理论顺序。
2. 将 **Flow Matching** 放到 **Diffusion Policy** 之后，使它成为生成式动作策略的自然延伸。
3. 将 **SSM / Mamba** 放到 **Transformer Policy** 与 **VLA** 之间，使其成为长序列策略骨架的平替与补充。
4. 将 **从论文到实机** 移动到最后，作为全书真正的终章。
5. 新增 **第23章 快慢模型**，把 VLA、World Model、ACT、Diffusion、Flow、SSM/Mamba、DPO 和部署组织成架构闭环。
6. 为全书新增本导读，并为七篇分别新增篇章导读。
7. 新增 **机械臂抓取与放置主线**，让全书从“方法清单”变成围绕同一机器人任务逐步展开的数学教程。

### 路径 A：完整教材路径

从第1章读到第29章。适合第一次系统学习模仿学习数学原理的读者。阅读时建议始终追问：这一章在机械臂抓取与放置任务中解决了哪种失败？


```text
第1章 → 第2章 → 第3章 → 第4章 → 第5章 → 第6章 → 第7章 → 第13章 → 第14章 → 第16章 → 第20章 → 第25章 → 第26章 → 第27章 → 第29章
```

适合想把本书内容落实到“视觉引导机械臂抓取与放置”项目中的读者。读这条路径时，应重点关注观测、动作空间、数据覆盖、闭环误差、动作多模态、动作块、实机安全和数据闭环。

```text

```

