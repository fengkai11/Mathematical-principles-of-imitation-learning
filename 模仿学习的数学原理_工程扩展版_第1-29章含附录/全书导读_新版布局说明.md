# 全书导读：新版布局说明

> 本版将原来的线性章节结构重排为 **八篇、二十九章、九个附录**。调整目标不是推翻原稿，而是把已经扩展出的内容重新组织成一条更清楚的技术演进路线。

## 1. 为什么要重新布局？

随着 Flow Matching、World Models / WAM / JEPA、SSM / Mamba、Preference Alignment / DPO 以及快慢模型被补入，全书已经不再只是一本“行为克隆到 Diffusion Policy”的入门教程，而是扩展为：

```text
模仿学习基础
→ 序列决策与轨迹分布
→ 分布匹配与奖励视角
→ 现代机器人策略模型
→ 长序列与多模态策略架构
→ 世界模型与快慢认知系统
→ 后训练、对齐与实机部署
```

如果继续按追加方式把新内容放在第20章之后，读者会觉得后半部分像“补丁”。新版布局改为分篇结构，目的是让每一组章节都有清晰的问题意识和数学承接。

## 3. 本次结构调整做了什么？

本次重排主要做了六件事：

1. 将 **IRL** 前移到 **GAIL** 之前，使第三篇形成“奖励反推 → 分布匹配 → 离线风险”的理论顺序。
2. 将 **Flow Matching** 放到 **Diffusion Policy** 之后，使它成为生成式动作策略的自然延伸。
3. 将 **SSM / Mamba** 放到 **Transformer Policy** 与 **VLA** 之间，使其成为长序列策略骨架的平替与补充。
4. 将 **从论文到实机** 移动到最后，作为全书真正的终章。
5. 新增 **第23章 快慢模型**，把 VLA、World Model、ACT、Diffusion、Flow、SSM/Mamba、DPO 和部署组织成架构闭环。
6. 为全书新增本导读，并为七篇分别新增篇章导读。
