# 第一篇数学主线增强执行计划

本文档用于落实“第一篇不从压缩角度，而从增强数学主线角度进行改造”的审稿意见。

本次改造不以删减篇幅为目标，而以建立更清晰的数学链路为目标：

```text
专家轨迹数据 D_E
→ 学习策略 pi_theta(a | o)
→ BC 在专家分布 d^{pi_E}(o) 上最小化单步损失
→ 部署时策略面对自己诱导出的分布 d^{pi_theta}(o)
→ 分布不一致导致误差累积
→ DAgger 用 d^{pi_i}(o) 上的新标注修正训练分布
```

---

## 一、改造原则

1. **不重写原章节主干**：保留第 1–4 章已有叙述、图示和阅读节奏。
2. **增强数学骨架**：在导读和章节增强页中补出关键公式、目标错配和符号关系。
3. **保持初学者友好**：公式后必须解释动机、符号、直觉和工程含义。
4. **避免符号混乱**：区分确定性策略、随机策略、专家动作标签、专家策略和访问分布。
5. **方便后续并回正文**：每个增强页都按“可直接并入原章”的小节组织。

### 2. 第 1 章从定义集合升级为数学对象建模

**目标**：第 1 章不仅定义观测、动作、轨迹和策略，还要说明这些对象如何组成模仿学习的建模闭环。

**应补内容**：

```math

```

```math
 D_E=\{\tau_i\}_{i=1}^N
```


以及：

```math
 \tau \sim p_\pi(\tau)
```

用于提前埋下策略诱导轨迹分布的伏笔。

**状态**：通过第 1 章增强页执行。

### 3. 第 2 章把 BC 目标写成专家分布上的经验风险

**目标**：第 2 章不只写有限样本平均损失，还要说明 BC 实际是在专家分布上优化单步模仿损失。

**应补内容**：

```math
\min_\theta
\mathbb{E}_{o\sim d^{\pi_E},\ a_E\sim\pi_E(\cdot\mid o)}
\left[
\ell\left(\pi_\theta(\cdot\mid o),a_E\right)
\right]
```

以及有限样本近似：

```math
\min_\theta
\frac{1}{N}\sum_{i=1}^N
\ell\left(\pi_\theta(\cdot\mid o_i),a_i\right)
```

**状态**：通过第 2 章增强页执行。

---

### 4. 第 2 章补充 MSE 是高斯 NLL 的特例

**目标**：把 MSE 与概率建模联系起来，为后续 CVAE、Diffusion Policy、Flow Matching 铺垫。

**应补内容**：

```math
\pi_\theta(a\mid o)=\mathcal{N}\left(a;f_\theta(o),\sigma^2I\right)
```

```math
-\log \pi_\theta(a\mid o)
=
\frac{1}{2\sigma^2}\|a-f_\theta(o)\|^2+C
```

**状态**：通过第 2 章增强页执行。

### 5. 第 3 章把分布偏移升级为训练目标与执行目标不一致

**目标**：让第 3 章从“现象解释”提升为“目标错配”。

**应补内容**：

```math
J_{BC}(\theta)
=
\mathbb{E}_{o\sim d^{\pi_E}}
\left[
\ell\left(\pi_\theta(o),a_E\right)
\right]
```

```math
J_{exec}(\theta)
=
\mathbb{E}_{o\sim d^{\pi_\theta}}
\left[
\ell\left(\pi_\theta(o),a_E\right)
\right]
```

并明确：

> 问题不是 loss 一定写错了，而是期望所在的分布变了。

**状态**：通过第 3 章增强页执行。

### 6. 第 3 章把 O(T^2 epsilon) 组织成小命题

**目标**：让误差累积公式更像教材中的数学命题，而不是孤立公式。

**应补内容**：

```text
命题：如果 BC 在专家分布上的单步错误率为 epsilon，任务 horizon 为 T，且错误会把策略带到训练数据覆盖不足的区域，则最坏情况下累计错误量级可能达到 O(T^2 epsilon)。
```

**状态**：通过第 3 章增强页执行。

---

### 7. 第 4 章补 DAgger 为什么改善误差累积的直觉

**目标**：第 4 章不只讲流程，还要接住第 3 章的 $O(T^2\varepsilon)$，解释 DAgger 的理论价值。

**应补内容**：

```text
BC：主要在 d^{pi_E} 上训练；
DAgger：逐轮把 d^{pi_i} 上的样本加入训练集；
直觉上可把长期错误从 O(T^2 epsilon) 改善到更接近 O(T epsilon)。
```

**状态**：通过第 4 章增强页执行。

### 8. 第 4 章补充混合策略公式的严谨解释

**目标**：避免读者误解 $\pi_i^{mix}=\beta_i\pi_E+(1-\beta_i)\pi_i$ 是直接平均两个动作。

**应补内容**：

```math
 a_t\sim \pi_i^{mix}(\cdot\mid o_t)
=
\beta_i\pi_E(\cdot\mid o_t)
+
(1-\beta_i)\pi_i(\cdot\mid o_t)
```

并解释：这是以概率选择专家策略或学习策略，或者在随机策略下形成混合分布。

**状态**：通过第 4 章增强页执行。
