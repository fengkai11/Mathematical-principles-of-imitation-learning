# 《模仿学习的数学原理》结构重排版说明

本版本是在“终稿扩展修订版（第1–24章含附录）”基础上完成的结构级整理版。

## 本版主要变化

1. 将全书重排为 **八篇、二十九章、九个附录**。
2. 将旧第14章 IRL 前移到 GAIL 之前，使理论顺序变为“奖励反推 → 分布匹配”。
3. 将旧第20章“从论文到实机”移动到最后，作为全书终章。
4. 将 Flow Matching 放到 Diffusion Policy 之后，作为生成式动作策略的自然延伸。
5. 将 SSM / Mamba 放到 Transformer Policy 与 VLA 之间，作为长序列架构平替与补充。
6. 新增第23章《快慢模型：通用机器人为什么需要双系统架构》。
7. 新增《全书导读：新版布局说明》以及八篇篇章导读。
8. 按新版章号重新统一了正文章节公式编号。
9. 同步更新了 SUMMARY、术语表、数学知识地图和章节目录。

## 推荐阅读入口

- 先读：`全书导读_新版布局说明.md`
- 再读：`全书章节目录_新版布局.md`
- 需要快速理解全书数学结构：`全书数学知识地图_新版布局.md`
