## 第 2 章：Behavior Cloning：最朴素，也最容易翻车的模仿学习

### 本章核心问题

为什么行为克隆可以看成监督学习？为什么训练起来像普通回归或分类，闭环执行时却可能翻车？

### 需要讲清楚的数学概念

- supervised learning；
- loss；
- 最大似然估计；
- 负对数似然；
- 交叉熵；
- MSE；
- 行为克隆目标：

\[
\theta^*
=
\arg\max_\theta
\sum_{t=1}^N
\log \pi_\theta(a_t|o_t)
\]

- 负对数似然形式：

\[
\mathcal L_{\mathrm{BC}}(\theta)
=
-\mathbb E_{(o,a)\sim\mathcal D}
[
\log \pi_\theta(a|o)
]
\]

### 推荐工程案例

- 机械臂动作回归；
- 自动驾驶方向盘角度预测；
- 泊车控制量模仿。

### 本章公式索引

- \(-\log \pi_\theta(a_t|o_t)\)
- \(\|a_t-\hat a_t\|^2\)
- \(\theta^*=\arg\max_\theta \sum_t \log \pi_\theta(a_t|o_t)\)

## 第 3 章：分布偏移：机器人为什么会一步错、步步错？

### 本章核心问题

为什么 BC 在训练集上很好，闭环执行时却越走越歪？训练时模型看到的是专家状态分布，执行时看到的却是自己制造出来的状态分布。

### 需要讲清楚的数学概念

- distribution shift；
- covariate shift；
- expert state distribution：

\[
d^{\pi_E}(s)
\]

- learned policy state distribution：

\[
d^{\pi_\theta}(s)
\]

- 训练目标与执行目标不一致：

\[
\mathbb E_{s\sim d^{\pi_E}}
[
\ell(\pi_\theta(s),\pi_E(s))
]
\]

vs.

\[
\mathbb E_{s\sim d^{\pi_\theta}}
[
\ell(\pi_\theta(s),\pi_E(s))
]
\]

- compounding error。

### 推荐工程案例

- 自动驾驶车道保持偏一点后越偏越远；
- 机械臂抓取偏一点后后续观测全变；
- 泊车第一把方向错了，后面全在补锅。

2. 小误差如何随时间累积；
3. open-loop loss 与 closed-loop failure 对比；
4. 自动驾驶偏离车道示意图。

### 本章公式索引

- \(d^{\pi_E}(s)\)
- \(d^{\pi_\theta}(s)\)
- \(\mathbb E_{s\sim d^{\pi_E}}[\cdot]\)
- \(\mathbb E_{s\sim d^{\pi_\theta}}[\cdot]\)
