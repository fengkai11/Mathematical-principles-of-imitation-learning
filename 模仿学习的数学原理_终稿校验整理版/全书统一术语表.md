# 全书统一术语表

> 本术语表用于统一全书中的常用概念、英文名称、中文译法和简要说明。阅读时遇到某个术语拿不准含义，可以先回到本表快速定位。

## 1. 基础对象与数据

| 中文术语 | 英文术语 | 建议记号 | 统一说明 |
|---|---|---|---|
| 状态 | State | \(s_t\) | 环境在时刻 \(t\) 的真实内部状态。 |
| 观测 | Observation | \(o_t\) | 智能体在时刻 \(t\) 实际能看到的输入，如图像、激光、关节状态。 |
| 动作 | Action | \(a_t\) | 策略输出的控制命令。 |
| 轨迹 | Trajectory | \(	au\) | 一次完整交互过程，通常写作 \((s_0,a_0,s_1,a_1,\dots,s_T)\)。 |
| 专家示范 | Expert Demonstration | \(\mathcal D_E\) | 由专家行为构成的数据集。 |
| 专家策略 | Expert Policy | \(\pi_E\) | 生成专家动作的策略。 |
| 数据集 | Dataset | \(\mathcal D\) | 用于训练或评估的样本集合。 |
| 动作块 | Action Chunk | \(a_{t:t+H-1}\) | 一次性预测的多步动作序列。 |
| 历史窗口 | History Window | — | 模型用于决策的历史观测和动作片段。 |

## 2. 策略与分布

| 中文术语 | 英文术语 | 建议记号 | 统一说明 |
|---|---|---|---|
| 策略 | Policy | \(\pi_	heta\) | 从状态或观测到动作分布的映射。 |
| 确定性策略 | Deterministic Policy | \(a=f_	heta(o)\) | 每个输入只输出一个动作。 |
| 概率策略 | Stochastic Policy | \(\pi_	heta(a\mid o)\) | 给定输入后输出动作分布。 |
| 条件分布 | Conditional Distribution | \(p(a\mid o)\) | 在给定条件下某个随机变量的分布。 |
| 轨迹分布 | Trajectory Distribution | \(p_\pi(	au)\) | 在策略 \(\pi\) 下生成整条轨迹的概率分布。 |
| 状态访问分布 | State Visitation Distribution | \(d_\pi(s)\) | 策略在执行中访问到各状态的频率分布。 |
| 占用测度 | Occupancy Measure | \(ho_\pi(s,a)\) | 状态—动作对在轨迹中的访问频率度量。 |
| 多模态动作分布 | Multimodal Action Distribution | — | 一个状态下可能存在多种都合理的动作模式。 |
| 分布偏移 | Distribution Shift | — | 训练分布和执行分布不一致。 |
| 协变量偏移 | Covariate Shift | — | 由于策略改变导致输入状态分布发生变化。 |

## 3. 损失函数与统计量

| 中文术语 | 英文术语 | 建议记号 | 统一说明 |
|---|---|---|---|
| 损失函数 | Loss Function | \(L(	heta)\) | 衡量模型预测与目标差异的函数。 |
| 最大似然估计 | Maximum Likelihood Estimation | MLE | 通过最大化观测数据出现概率来估计参数。 |
| 负对数似然 | Negative Log-Likelihood | NLL | 最大似然常用等价损失形式。 |
| 交叉熵 | Cross Entropy | — | 离散动作常用监督损失。 |
| 均方误差 | Mean Squared Error | MSE | 连续动作回归的常见损失。 |
| KL 散度 | Kullback–Leibler Divergence | \(D_{KL}(p\|q)\) | 衡量两个分布差异的常用量。 |
| 熵 | Entropy | \(H(p)\) | 衡量分布不确定性的指标。 |
| 最大熵原理 | Maximum Entropy Principle | — | 在约束下选择最不额外做假设的分布。 |
| ELBO | Evidence Lower Bound | ELBO | VAE/CVAE 中用于训练的下界目标。 |
| Score Function | Score Function | \(
abla_x \log p(x)\) | 分布对输入的对数密度梯度。 |
| Score Matching | Score Matching | — | 通过匹配 score 来学习分布的方法。 |

## 4. 经典模仿学习方法

| 中文术语 | 英文术语 | 统一说明 |
|---|---|---|
| 行为克隆 | Behavior Cloning, BC | 把模仿学习视为监督学习，直接拟合专家动作。 |
| 数据聚合 | Dataset Aggregation, DAgger | 让当前策略自己跑，再让专家补标，从而修复分布偏移。 |
| 逆强化学习 | Inverse Reinforcement Learning, IRL | 从专家行为中反推潜在奖励函数。 |
| 最大熵逆强化学习 | Maximum Entropy IRL | 假设专家轨迹服从最大熵形式的分布。 |
| 生成式对抗模仿学习 | Generative Adversarial Imitation Learning, GAIL | 通过对抗训练直接匹配专家与策略行为分布。 |
| 离线模仿学习 | Offline Imitation Learning | 只用固定离线数据训练策略，不再与环境在线交互。 |

## 5. 现代策略建模方法

| 中文术语 | 英文术语 | 统一说明 |
|---|---|---|
| 条件变分自编码器 | Conditional Variational Autoencoder, CVAE | 用隐变量建模动作多样性与风格。 |
| 动作块变换器 | Action Chunking with Transformers, ACT | 用 Transformer 一次预测一段动作。 |
| 扩散策略 | Diffusion Policy | 用扩散生成过程建模复杂动作分布。 |
| 决策 Transformer | Decision Transformer | 把决策建模写成序列建模问题。 |
| Transformer 策略 | Transformer Policy | 用自注意力统一建模历史上下文和动作预测。 |
| 视觉—语言—动作模型 | Vision-Language-Action, VLA | 将视觉、语言和动作统一到同一大模型框架中。 |
| 世界模型 | World Model | 显式或隐式学习环境动态与可预测结构。 |

## 6. 机器人与应用语境

| 中文术语 | 英文术语 | 统一说明 |
|---|---|---|
| 开环评测 | Open-loop Evaluation | 不让策略真正闭环控制，只离线比较预测结果。 |
| 闭环执行 | Closed-loop Execution | 策略输出会影响后续状态，需要真实滚动执行。 |
| 失效恢复 | Recovery | 系统偏离理想状态后重新回到可控轨道的能力。 |
| 安全约束 | Safety Constraint | 在实机部署中对动作、状态和系统行为施加的限制。 |
| 回退机制 | Fallback | 当策略不可靠时切回传统控制器或安全模式。 |
| 数据覆盖度 | Coverage | 训练数据对真实任务空间的覆盖范围。 |
| 分布外输入/动作 | Out-of-Distribution, OOD | 超出训练数据支持范围的输入或动作。 |

## 7. 使用建议

1. 如果某一章的数学推导看起来复杂，优先回到本术语表确认对象、分布和损失分别在说什么。  
2. 阅读现代方法（ACT、Diffusion Policy、VLA）时，先抓住“它是在建模什么分布”，再看网络结构。  
3. 术语表的目的不是替代正文，而是帮助全书在概念层面保持统一语言。
